Visualization Package Descriptions

  • Altair: A declarative statistical visualization library for Python, built on Vega and Vega-Lite.
  • Altair-Viewer: A viewer for Altair visualizations, enabling quick rendering of charts.
  • Bokeh: A Python library for creating interactive visualizations for modern web browsers.
  • Boruta: A feature selection method for machine learning, wrapping around random forests.
  • Cartopy: A cartographic library for geospatial data processing and mapping in Python.
  • Chartify: An easy-to-use Python charting library built on top of Bokeh.
  • Cufflinks: Connects plotly with pandas to create interactive visualizations directly from dataframes.
  • Dash: A Python framework for building analytical web applications.
  • Dash-Bootstrap-Components: A library for adding Bootstrap components to Dash applications.
  • Datawrapper: A tool for creating interactive charts, maps, and tables.
  • Datarobot: A machine learning platform providing tools for automated model building and deployment.
  • Datashader: A data visualization library designed for large datasets.
  • Echarts: A powerful visualization library built on JavaScript, with Python bindings through `pyecharts`.
  • Folium: A Python library for visualizing data on interactive leaflet maps.
  • Geemap: A Python package for interactive mapping with Google Earth Engine.
  • Geopandas: Extends pandas to handle geospatial data, making it easier to work with geographic information.
  • Geoviews: A Python library for geographic data visualization, extending HoloViews.
  • Geoplotlib: A toolbox for plotting geographical data and creating interactive maps.
  • Glumpy: A Python library for scientific visualization on modern graphics hardware.
  • Holoviews: A high-level library for data analysis and visualization in Python, built on Bokeh and Matplotlib.
  • Hvplot: Provides a high-level plotting API for the PyData ecosystem, built on HoloViews.
  • Ipyleaflet: A Jupyter widget for interactive maps using Leaflet.js.
  • Ipyvolume: A 3D plotting library for the Jupyter notebook, supporting volume rendering.
  • Ipywidgets: A framework for building interactive HTML widgets in Jupyter notebooks.
  • Joypy: A Python package for creating joyplots (overlapping density plots).
  • K3D: A Jupyter notebook 3D visualization package for scientific computing and data visualization.
  • Keplergl: An open-source geospatial analysis tool, developed by Uber, for exploring large-scale datasets.
  • Lazypredict: A package to build multiple machine learning models with minimal code.
  • Leafmap: A Python package for interactive mapping and geospatial data analysis.
  • Lumino: A component toolkit used by JupyterLab for building user interfaces.
  • Matplotlib: A foundational plotting library for Python, providing 2D and 3D plotting capabilities.
  • Mayavi: A scientific data visualization tool, supporting 3D plotting, built on VTK.
  • Mizani: A scales package for the plotnine graphics library, extending ggplot2's grammar of graphics.
  • Mpld3: A toolkit for exporting Matplotlib plots to interactive, D3-powered HTML.
  • Mplfinance: A package for visualizing financial data with Matplotlib.
  • Networkx: A Python package for creating, analyzing, and visualizing complex networks.
  • Nbformat: The reference implementation of the Jupyter Notebook format.
  • Orange3: A suite of machine learning and data visualization tools, focusing on visual programming.
  • Panel: A Python library for creating interactive dashboards and web apps with Python.
  • Pandas-Bokeh: A library for plotting pandas data using Bokeh, enabling interactive visualizations.
  • Phik: A package for calculating the phi_k correlation coefficient for data associations.
  • Plotly: A popular interactive graphing library for Python, supporting various types of plots.
  • Plotly-Express: A high-level interface to Plotly for creating quick and easy visualizations.
  • Plotnine: A Python implementation of ggplot2's grammar of graphics for creating complex visualizations.
  • Pthreejs: A Python binding for the Three.js 3D visualization library.
  • Pydantic: A data validation library used for defining structured, type-annotated data.
  • Pyecharts: A Python library binding for the Echarts visualization library.
  • Pygal: A Python library for creating SVG charts.
  • Pyvis: A Python library for interactive network visualization, using Vis.js.
  • PyVista: A 3D plotting and mesh analysis package for Python.
  • Pywaffle: A library for creating "waffle" charts in Python.
  • Pydeck: A Python wrapper for deck.gl, enabling geospatial visualizations in Python notebooks.
  • Qgrid: An interactive grid for Jupyter notebooks, allowing manipulation of pandas dataframes.
  • Scattertext: A tool for creating interactive, term-frequency scatter plots in Python.
  • Scikit-Plot: A plotting library for scikit-learn, enabling visualizations for machine learning.
  • Seaborn: A Python visualization library based on Matplotlib, providing high-level interface for statistical graphics.
  • Streamlit: A fast way to build and share data apps using Python.
  • Toyplot: A modern, interactive plotting library aimed at scientific visualization.
  • Vaex: A Python package for high-performance dataframes, enabling large-scale data visualization and exploration.
  • Vega: A visualization grammar for creating, sharing, and exploring interactive visualizations.
